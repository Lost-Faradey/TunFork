# ⚡ TunFork

AmneziaWG tunnel management utility with flexible traffic routing.

## Features

- 🎮 **Split tunneling** — selected services via tunnel, games direct
- 🛡 **Auto-heal** — monitors tunnel and recovers on failure
- 🔍 **Leak detector** — finds IPs bypassing the tunnel
- 🔑 **Licensing** — 3-day trial via Telegram bot @TunFork_bot

## Requirements

- Windows 10/11
- [AmneziaWG](https://amnezia.org) (install separately)
- Your own WireGuard config

## Installation

1. Download `TunFork_v1.0.zip` from [Releases](https://github.com/LostFaradey/TunFork/releases)
2. Extract to any folder
3. Copy your `.conf` to `data/base_profile.conf`
4. Run `main.exe` (as Administrator)
